# gf2vs
gf2vs is a go package implementing a vector space for bit vectors of arbitrary length.
Functions of the vector space are in the file gf2vs.go.
Functions supporting bit matrixes of any size are in bitmatrix.go. 
In this file functions for computing a solution of an XOR-SAT via Gaussian elimination
are provided.
